Install guide

How to open and use the extension in Chrome.

1. Unzip the download

Extract the Send-Notes-Extension.zip file to a folder you can keep on your computer.

2. Load unpacked

Open chrome://extensions, turn on Developer mode, then choose Load unpacked and select the extracted folder.

3. Start highlighting

Open any webpage, highlight text, save it into the extension, and generate cohesive notes from your selected passages.

User manual

Quick reference for the Send Notes workflow.

A

Capture highlights

Highlight a sentence or paragraph and save it through the right-click menu or the extension popup.

B

Add your API key

Inside the popup, open AI settings, enter your OpenAI API key, pick a model, and save.

C

Generate notes

Use Generate AI note to turn your chosen highlights into a clearer, more cohesive note set.
